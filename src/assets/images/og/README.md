# OG Images

Place final 1200Ă—630px WebP OG images here for each page type.

## Specs
- **Dimensions**: 1200Ă—630px (ratio 1.91:1)
- **Format**: WebP
- **Max size**: < 2 MB
- **File naming**: `{page-type}.webp`

## Required Files

| File | Page |
|------|------|
| `home.webp` | Homepage / |
| `about.webp` | /tentang-kami |
| `packages.webp` | /paket-wisata |
| `destinations.webp` | /destinasi |
| `reviews.webp` | /ulasan |
| `guides.webp` | /panduan-wisata |
| `contact.webp` | /kontak |
| `faq.webp` | /faq |
| `accommodations.webp` | /akomodasi |
| `car-rental.webp` | /sewa-mobil |
| `privacy.webp` | /kebijakan-privasi |
| `terms.webp` | /syarat-ketentuan |
| `blog.webp` | /blog/* |
| `package-detail.webp` | /paket-wisata/*/*/\* |
| `destination-detail.webp` | /destinasi/\* |
| `vehicle-detail.webp` | /sewa-mobil/*/\* |

## Special Cases
The following region collections in paket-wisata use local-region-specific OG images and are excluded from the redirect-to-default-image logic:
- `open-trip-lombok-3d2n`
- `open-trip-lombok-4d3n`
